<title>

Specifies the title of the web page; title text appears in the browser title bar and as the bookmark or favorites text: Global attributes.

<script>

Contains dynamic script or data content. Global attributes, plus: type (MIME), src (address and the external scripts), defer (if set to defer the script is executed after the page is loaded, charset (character encoding of the external script).

<style>

Used in the "head" section to contain CSS style rules. Attributes: type (valid MIME), media (valid media query type)

<body>

Identifies the body section of the web page

<p>

Marks the beginning of a new block of text. Global attribute.

division

Indicates a division within a document. Global attributes.

<blockquote>

Indents text on both the left and right margins.

<td>

<section>

The section of a document typically including a title or heading. Global attributes.

<article>

A section of content that forms an independent part of a document or web site. Global attributes.

<nav>

Group of navigation links. Global attributes.

<h1> through to <h6>

Defines headings for six levels of content. Global attribute.

<header>

Header of a section. Global attribute.

<em>

Emphasizes text, usually as italic; browser determines the text style. Global attributes.

<strong>

Contains important text, usually displayed as boldface; browser determines the text style. Global attributes.

<span>

<img>

Inserts an image into a web page. This is an empty element. Global attributes, plus: width, height.

<audio>

Contains an audio stream. Attributes; src (url) autoplay, preload, controls (display playback controls), loop (video plays repeatedly)

<video>

Video or movie content. Global attributes, plus: autoplay, preload, controls, loop, poster, height, width.

<a>

Creates a clickable hypertext anchor in a document; can be text or image. Global attributes, plus: href, name, target window or frame, hreflang, rel (defines the relationship between doc and destination), media, type (MIME)
